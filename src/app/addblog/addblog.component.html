Writing Instructions
  • Give the title to your story
  • Add heading using the heading tab
  • You can change font-family (It is limited to 3 as of now)
  • You can add bold texts using B option from toolbar
  • You can also add Italic text and also underline the texts
  • Colors, to change text or background of text.
  • Align the text using alignment tab
  • Add links by selecting a word
  • You can also add images to your story using the image option
  • To add code, select code otpion from toolbar
  • Spelling mistakes? Not anymore with the spell checker.
