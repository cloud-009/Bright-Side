Writing Instructions
  • Give the title to your story
  • For heading, choose the first option from toolbar and you can add heading and subheading
  • You can change the fon-family with the second option. (It is limited to 3 as of now)
  • You can add bold texts using B option from toolbar
  • You can also add Italic text and also underline the texts
  • Colors, to change text or background of text.
  • Alignment of text can also be done using the aligning options from toolbar
  • To add links, type the link and then select the typed link and click the link option from toolbar
  • You can also add images to your story using the image option
  • To add code, select code otpion from toolbar and either you can type or paste the code from your code-editor
  • Spelling mistakes? Not anymore with the spell checker.
